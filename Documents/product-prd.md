# product-prd
- **Purpose**: Capture vision, goals, and success metrics.
- **Scope**: Superteam Academy experience for Solana devs (PT-BR/ES/EN).
- **Read when**: Prioritizing features or aligning roadmap.

## Vision
Open, on-chain LMS that takes builders from zero to deploying production-ready Solana dApps with interactive challenges and verifiable credentials.

## Goals
- Interactive, project-based learning with code runner.
- On-chain XP and evolving cNFT credentials.
- Gamification: streaks, achievements, leaderboard.
- Multilingual (PT-BR, ES, EN) from day one.

## Success metrics
- Activation: first lesson + challenge completion.
- Retention: 7/30-day streaks, module completion rate.
- Credential issuance: cNFTs minted/upgraded.
- Engagement: challenges attempted, hints used, leaderboard participation.

## Audiences
- New Solana devs, existing web3 devs, LATAM builders.

## Scope hints
- Stub backend flows now; swap in on-chain later.
- Wallet linking required to finalize credentials.
